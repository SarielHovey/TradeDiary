# TradeDiary
A trading diary tool
## 描述
个人自用的交易日志工具, 收盘后用于记录复盘的想法和对下个交易日的计划<br>
没有多少技术含量, 过于落后, 方便开源<br>

## 使用环境要求
首先, 需要`Python ver >= 3.9`<br>
其次, 本工具完全基于Python内置包`json`; 如果`import json`发生错误, 建议反思一下Python按照是否正确<br>
## 数据结构
本工具最终会输出并管理存放在本地的`json`文件, 文件结构如下:
```
施工中...
```
